# Dropdown-list-animations

## General Information
Next project focusing on developing CSS skills. Each menu is unroll in a different, creative way.
<br>
The menu items use features such as:
- animation-delay,
- transform: rotate(),
- ransform: translateY(), 
- transform: scale().

## Technologies Used
- HTML version: 5
- CSS version: 3 and SASS
